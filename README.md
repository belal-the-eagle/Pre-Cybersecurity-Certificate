# Pre-Cybersecurity-Certificate
🎯 الحمد لله، خطوة أولى رسمية في طريق الأمن السيبراني! 🛡️
سعيد جداً بإنهاء كورس Pre-Cybersecurity من أكاديمية Netriders وحصولي على الشهادة الخاصة به. الكورس كان فرصة ممتازة لتثبيت المفاهيم الأساسية وفهم البنية التحتية للأنظمة والشبكات بشكل سليم.
دي البداية وبس، وخريطة الطريق واضحة قدامي للخطوات الجاية (Linux Terminal ➡️ Network ➡️ Security+ ➡️ eJPT).
شكراً لكل حد بيدعم ويشارك مصادر مفيدة لتسهيل الطريق على الشباب والمبتدئين في المجال. القادم أقوى بإذن الله! 🚀💻
